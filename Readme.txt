----- Note Please Install NPM and Node.js before running scrips -----------

$ npm install express --save


The given folder contains code to run a web application of two pages on a server.
---------------------------------------------------------------------------



---------------------- Program Run Instructions----------------------------

Step 1: Verify the current folder on Terminal to be ./FizzBuzz_Counter

$ node Server_Setup.js

---------------------------------------------------------------------------
